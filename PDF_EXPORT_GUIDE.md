## 🎯 功能概述

### 什么是 PDF 覆盖导出？

在**原文 PDF 上直接覆盖翻译文本**，完美保留原排版！

### 对比其他方案

| 方案 | 优点 | 缺点 |
|------|------|------|
| **PDF 覆盖（当前）** | ✅ 保留原排版<br>✅ 双栏/图片/公式都对齐<br>✅ 类似小绿鲸效果 | ⚠️ 翻译文本在页面底部<br>⚠️ 中文字体依赖系统 |
| Word 导出 | ✅ 可编辑<br>✅ 格式通用 | ❌ 不保留排版<br>❌ 双栏变单栏 |
| 纯文本导出 | ✅ 简单快速 | ❌ 无任何格式 |

---

## 🚀 使用方法

### 步骤 1：上传 PDF
1. 访问 http://localhost:3000/translate
2. 点击「📎 上传 PDF」
3. 选择论文文件（最大 10MB）

### 步骤 2：翻译
1. 选择学科领域和翻译模式
2. 点击「开始翻译」
3. 等待翻译完成

### 步骤 3：导出 PDF
1. 在导出格式中选择「📕 PDF」
2. 点击「📕 PDF」导出按钮
3. 自动下载翻译后的 PDF

## 🔧 技术实现

### 核心库

```javascript
import { PDFDocument, rgb, StandardFonts } from 'pdf-lib';
```

### 工作流程

1. **加载原 PDF**
   ```javascript
   const pdfDoc = await PDFDocument.load(uint8Array);
   ```

2. **分割翻译文本**
   ```javascript
   const charsPerPage = 500;
   const textPages = translatedText.split(charsPerPage);
   ```

3. **在每页添加翻译**
   ```javascript
   // 绘制浅蓝色背景
   page.drawRectangle({
     color: rgb(0.95, 0.95, 1.0),
     opacity: 0.8,
   });

   // 添加翻译文本
   page.drawText(translatedText, {
     size: 10,
     color: rgb(0, 0, 0.5),
   });
   ```

4. **保存并返回**
   ```javascript
   const pdfBytes = await pdfDoc.save();
   ```

## 📝 当前实现细节

### 翻译文本位置
- **起始位置**: 距页面顶部 150px
- **字体大小**: 10pt
- **行高**: 12px
- **背景色**: 浅蓝色 (RGB: 0.95, 0.95, 1.0)
- **文字色**: 深蓝色 (RGB: 0, 0, 0.5)
- **标注**: `[翻译]` 灰色小字

### 分页策略
- **每页字符数**: 约 500 字符
- **超出处理**: 自动添加新页面
- **页数不足**: 翻译文本多时自动加页

- 浅蓝色半透明背景（80% 不透明）
- 与原文区分明显
- 不影响阅读原文

### 文字标注
- `[翻译]` 灰色小字标注
- 位置：每页右上角
- 避免与原文混淆

## ⚠️ 已知限制

### 当前版本
1. **翻译位置固定**: 都在页面底部
2. **无法精确对齐**: 不能对应到具体段落
3. **字体单一**: 使用标准 Helvetica 字体
4. **不支持中文嵌入字体**: 可能需要额外配置

## 📊 与 Word 导出对比

| 场景 | 推荐格式 | 原因 |

| **阅读论文** | 📕 PDF | 保留原排版，方便对照 |
| **编辑翻译** | 📄 Word | 可直接修改文字 |
