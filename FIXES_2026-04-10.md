### 3. ✅ PDF 上传解析功能修复

**问题**:
- CJK 文字（中文/日文/韩文）可能解析失败
- 缺少文件验证
- 错误处理不完善
- 失败后未清理临时文件

**修复**:

#### 3.1 增强 `lib/pdfLayout.js`
- ✅ 添加 CMap 支持（改善 CJK 文字提取）
- ✅ 改进 Buffer 处理（兼容 Uint8Array）
```javascript
cMapUrl: 'https://cdnjs.cloudflare.com/ajax/libs/pdf.js/3.11.174/cmaps/',
cMapPacked: true,
```

#### 3.2 增强 `pages/api/upload-pdf-advanced.js`
- ✅ 添加 PDF 文件头验证（`%PDF` 检查）
- ✅ 添加文件大小限制（10MB）
- ✅ 改进 base64 解码错误处理
- ✅ 失败后自动清理临时文件
- ✅ 添加详细错误信息

**新增验证**:
```javascript
// PDF 文件头验证
function isValidPdf(buffer) {
  if (!buffer || buffer.length < 4) return false;
  const header = buffer.slice(0, 4).toString('ascii');
  return header === '%PDF';
}
```
