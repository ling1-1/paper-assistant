### 2. PDF 排版保留

#### 普通版 vs 增强版

**普通版** (`upload-pdf.js`):
- ✅ 快速提取
- ✅ 适合单栏 PDF
- ❌ 双栏排版会错乱

**增强版** (`upload-pdf-advanced.js`):
- ✅ 识别双栏排版
- ✅ 按位置重组文本
- ✅ 保留段落结构
- ✅ 逐页提取
- ⚠️ 稍慢（多 1-2 秒）

#### 增强版工作原理

```
原始 PDF（双栏）:
┌────────────┬────────────
│ Column A1  │ Column B1  │
│ Column A2  │ Column B2  │
└────────────────────────┘

普通版提取:
A1 B1 A2 B2 (混乱)

增强版提取:
1. 按 Y 坐标分行
2. 每行内按 X 坐标排序（左→右）
3. 合并：A1 B1
         A2 B2
4. 保留段落分隔
```

## 🔧 技术细节

### 翻译 Prompt 优化

```javascript
【重要】保留所有 LaTeX 公式（$...$, $$...$$, \ce{}, \frac{}, \alpha 等）
【重要】保留引用编号（如 [1], [2], [10-15]）、图表编号

保留段落结构，按原文分段输出
```

### PDF.js 增强提取

```javascript
// 1. 提取文本项及其位置
items.map(item => ({
  str: item.str,
  x: item.transform[4],  // X 坐标
  y: item.transform[5],  // Y 坐标
}));

// 2. 按 Y 坐标分行
rows.find(row => Math.abs(row.y - item.y) < 5);

// 3. 每行内按 X 排序
row.items.sort((a, b) => a.x - b.x);

// 4. 拼接成文本
row.items.map(item => item.str).join(' ');
```

##  已知限制

### 增强版 PDF 解析

1. **复杂表格**: 可能仍会错乱
2. **图片中的文字**: 无法提取（需要 OCR）
3. **手写公式**: 无法识别
4. **加密 PDF**: 不支持

### 翻译质量

1. **超长公式**: 可能被截断
2. **罕见符号**: 可能识别错误
3. **多语言混合**: 需要手动调整
