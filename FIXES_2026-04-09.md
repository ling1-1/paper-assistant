## 🐛 用户反馈的问题

### 1. 模式切换 UI 不变
**问题**: 切换翻译/术语解释/润色模式时，输入框和输出框的标题和提示不变

**修复**:
- ✅ 添加 `getModeHint()` - 根据模式动态显示提示
- ✅ 添加 `getModeTitle()` - 根据模式显示输入框标题
- ✅ 添加 `getOutputTitle()` - 根据模式显示输出框标题

**效果**:
- 翻译模式：「英文原文」→「中文翻译」
- 术语解释：「待解释内容」→「术语解释」
- 润色模式：「英文原文」→「中文润色」

### 2. PDF 上传报错
**错误信息**:
```
TypeError: (0 , pdf_parse__WEBPACK_IMPORTED_MODULE_3__.default) is not a function
```

**原因**: pdf-parse 的导入方式不兼容 Next.js 的 Webpack 打包

**修复**:
```javascript
// 修复前
import pdfParse from 'pdf-parse';

// 修复后
import pdfParse from 'pdf-parse/lib/pdf-parse.js';
```

**验证**: 重启服务后测试 PDF 上传
